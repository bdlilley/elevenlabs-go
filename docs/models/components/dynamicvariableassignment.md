# DynamicVariableAssignment

Configuration for extracting values from tool responses and assigning them to dynamic variables.


## Fields

| Field                                                                                                                                                          | Type                                                                                                                                                           | Required                                                                                                                                                       | Description                                                                                                                                                    |
| -------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `Source`                                                                                                                                                       | `*string`                                                                                                                                                      | :heavy_minus_sign:                                                                                                                                             | The source to extract the value from. Currently only 'response' is supported.                                                                                  |
| `DynamicVariable`                                                                                                                                              | `string`                                                                                                                                                       | :heavy_check_mark:                                                                                                                                             | The name of the dynamic variable to assign the extracted value to                                                                                              |
| `ValuePath`                                                                                                                                                    | `string`                                                                                                                                                       | :heavy_check_mark:                                                                                                                                             | Dot notation path to extract the value from the source (e.g., 'user.name' or 'data.0.id')                                                                      |
| `Sanitize`                                                                                                                                                     | `*bool`                                                                                                                                                        | :heavy_minus_sign:                                                                                                                                             | If true, this assignment's value will be removed from the tool response before sending to the LLM and transcript, but still processed for variable assignment. |
