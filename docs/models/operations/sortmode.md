# SortMode

Sort mode for listing tests. Use 'folders_first' to place folders before tests.

## Example Usage

```go
import (
	"github.com/bdlilley/elevenlabs-go/models/operations"
)

value := operations.SortModeDefault
```


## Values

| Name                   | Value                  |
| ---------------------- | ---------------------- |
| `SortModeDefault`      | default                |
| `SortModeFoldersFirst` | folders_first          |
